REFLEXIÓN FIN DE CURSO

¿Qué aprendí?

En el curso de Desarrollo de Aplicaciones Web, se adquirió una comprensión profunda de las tecnologías web, abarcando tanto los aspectos teóricos como prácticos. Se aprendieron los fundamentos de la web, incluyendo el funcionamiento de DNS, TCP/IP y HTTP, así como los roles específicos en el desarrollo web como desarrollador frontend, backend y fullstack. Se exploraron técnicas y herramientas para el desarrollo frontend, utilizando lenguajes como HTML y CSS, y frameworks como Bootstrap y Tailwind para diseñar interfaces de usuario. Además, se desarrollaron habilidades en JavaScript y el uso de frameworks modernos como React, enfatizando el manejo del DOM, la creación de componentes y el uso de hooks para la gestión del estado. En el ámbito backend, se abordó el desarrollo de aplicaciones web completas utilizando Next.js, incluyendo el consumo de APIs y la implementación de autenticación. También se estudió el desarrollo de APIs RESTful con Spring Boot, proporcionando una visión completa de la creación de soluciones tecnológicas robustas y escalables.

¿Cómo aprendí?

El proceso de aprendizaje en el curso fue facilitado a través de una combinación efectiva de clases teóricas y prácticas. Las sesiones teóricas proporcionaron los fundamentos necesarios, mientras que las sesiones prácticas en laboratorio permitieron aplicar los conocimientos adquiridos en tiempo real. Se utilizaron plataformas virtuales como Microsoft Teams para acceder a materiales de estudio . La participación en proyectos prácticos fue un componente clave, permitiendo a los estudiantes aplicar conceptos y técnicas en la creación de aplicaciones web reales. Además, la interacción constante con el profesor y compañeros de clase ofreció oportunidades para resolver dudas y recibir retroalimentación, enriqueciendo así el proceso de aprendizaje.
